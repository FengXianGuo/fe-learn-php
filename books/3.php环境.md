# php语法

## 一，标识符

<?php?>为php的语法的标识符

标识符的作用是让apache识别为php代码

## 二，以“;”为结束标记

每行均以;号结尾

parse error : 语法错误

## 三，函数

创建函数

    function add($a,$b){
        return $a+$b;
    }

使用函数

    echo add(1,2);
    
## 四，php和html的混编

    <body>
    <?php
        echo "hello wor22";
        $name = "123";
        echo $name;
        echo add(1,2);
        function add($a,$b){
            return $a+$b;
        }
    ?>
    </body>
    
php代码直接放在body里即可

## 五，创建数组


创建一个键为nav，值为“首页”的数组（此数组非js中的数组）


    <?php
        $arr = array("nav"=>"首页");
    ?>
